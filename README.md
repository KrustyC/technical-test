# Technical Test

# Installation
To install type the following: 
```
$> composer.phar install

$> npm install 

$> bower install
```

# Usage

To run the test type the following:
```
$> cd public
$> php -S localhost:8080

```
Open a web browser and insert the following URL: 
`localhost:8080`

# Testing
To run the test type the following:
`$> ./app/vendor/bin/phpunit`

## Requirements

- PHP v5.6
- SLIM v3.7
- PHPUnit v5.6
